## 📋 Overview

Career AI Agent is an advanced, conversational chatbot designed to be your personal career counselor. Powered by Google's **Gemini 2.5 Flash** model, it provides personalized, context-aware advice. Unlike simple Q&A bots, it remembers your conversation history and tailors responses based on your profile, offering a natural and intelligent guidance experience.

## ✨ Key Features

- **🗣️ True Conversational AI**: Engages in natural, flowing conversations. It remembers the last 3-4 exchanges to provide coherent follow-up answers.
- **👤 Personalized Profile**: An optional sidebar lets you share your name, current role, target role, experience, and location. The agent uses this information to personalize every response.
- **💡 Context-Aware Memory**: Maintains conversation context. It can refer back to your previous questions and answers, just like a human counselor.
- **💰 Intelligent Responses**: Delivers clear, concise, and practical information on:
  * Salary expectations (in INR with location & experience factors)
  * Required skills and technologies
  * Interview preparation tips and common questions
  * Daily work and career growth paths
- **🎨 Professional UI**: Features a sleek, modern interface with:
  * Gradient headers and glass-morphism effects
  * Distinct user and assistant chat bubbles with avatars
  * An informative welcome screen with feature highlights
  * A beautifully styled sidebar for profile management
- **⚡ Graceful Handling**: Politely handles off-topic questions, vague inputs, or casual greetings by steering the conversation back to career guidance.

## 📊 How It Works

| Step | Description |
|------|-------------|
| **1. Start** | Upon first visit, a welcoming screen introduces the agent's capabilities |
| **2. Profile (Optional)** | Use the sidebar to input your career details for personalized advice |
| **3. Ask** | Type your career-related question in the chat input |
| **4. Response** | Agent analyzes your question, considers profile & history, generates helpful response |
| **5. Continue** | Ask follow-up questions - the agent remembers context for smooth conversation |
