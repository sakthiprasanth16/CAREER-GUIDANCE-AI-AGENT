## 📋 Overview

Career Guidance AI Agent is an intelligent chatbot powered by Google's Gemini 2.5 Flash model that provides personalized career counseling. It helps users get detailed information about any career path including required skills, education, responsibilities, interview preparation, learning resources, and salary expectations in Indian Rupees (INR).

## ✨ Features

- ✅ **Interactive Chat Interface** - Natural conversation flow
- ✅ **Personalized Guidance** - Tailored advice based on your name and target role
- ✅ **Complete Career Roadmap** - Get all information in one place
- ✅ **Indian Salary Ranges** - Salary information in INR for Indian market
- ✅ **Quick Action Buttons** - Instant answers for common questions
- ✅ **Follow-up Questions** - Ask anything after receiving guidance
- ✅ **Clean Professional UI** - Modern, responsive design
- ✅ **No File Uploads** - Pure chat experience

## 📊 What Information You Get

| Section | Details |
|---------|---------|
| 🎯 **Required Skills** | Technical skills (8-10) + Soft skills (5) with complete explanations |
| 📚 **Education** | Minimum requirements, recommended degrees, alternative paths |
| 💼 **Key Responsibilities** | Daily tasks, role explanation, typical day activities |
| 🎯 **Interview Guide** | 5-step complete interview preparation |
| 📖 **Learning Resources** | Best courses, books, practice platforms |
| 💰 **Salary (INR)** | Entry, Mid, Senior level ranges for Indian market |
